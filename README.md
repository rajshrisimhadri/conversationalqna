# Conversational Q&A Chatbot 

This project is a Conversational Question & Answer Chatbot built using Python and LangChain.  
It allows users to ask questions from documents and receive intelligent, context-aware answers using large language models.

The chatbot is designed to demonstrate how to build document-based conversational AI systems using modern LLM frameworks.

---

##  Features

- Conversational Q&A interface  
- Document-based question answering  
- Context-aware responses  
- Built with LangChain  
- Easy to extend for new data sources  

---

##  Tech Stack

- Python  
- LangChain  
- OpenAI / LLM APIs  
- Streamlit (if applicable)  
- Vector databases (FAISS / Chroma if used)
